#### Prerequisites
You will need:
* Python 3.7+
* Pipenv
* An active internet connection


To install all the dependencies and start the virtual env, run the commands:
```sh
$ pipenv install
$ pipenv shell
```



---
#### Usage
```sh
$ python3 bs_scraper.py
```
